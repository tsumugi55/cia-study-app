# CIA直前確認ノート

APIを使わない、サーバー不要の学習ログ保存アプリです。

## 使い方

1. ChatGPTなどのAIに、その日の問題演習で分からなかった点を複数聞きます。
2. 学習終了時に、このアプリの「依頼文をコピー」を押します。
3. コピーした依頼文をChatGPTに送信し、整理済みJSONを作ってもらいます。
4. 返ってきたJSONをこのアプリに貼ります。
5. 「取り込みプレビュー」でPartや分野の振り分けを確認します。
6. 「取り込んで保存」を押します。
7. 「分野別確認資料」「学習まとめ」「直前確認モード」で見返します。

未整理のチャット履歴をそのまま貼った場合、アプリは無理に分割せず「未整理ログ」1件として保存します。

## 保存先

データはブラウザのlocalStorageに保存されます。サーバーやAPI料金は不要です。

Safariのデータ削除や機種変更で消える可能性があるため、「JSONエクスポート」で定期的にバックアップしてください。

## スマホで使う場合

おすすめは、`outputs` フォルダーの中身を静的サイトとして公開し、スマホのブラウザで開く方法です。

### 一番安定する方法

1. `outputs` フォルダーの中身をGitHub Pages、Netlify、Vercelなどの静的サイトに置きます。
2. 発行されたHTTPSのURLをスマホで開きます。
3. iPhone Safariの場合は共有メニューから「ホーム画面に追加」を選びます。
4. 以後はホーム画面のアイコンから開きます。

### 注意

- データはそのスマホのブラウザ内に保存されます。
- 別のスマホや別ブラウザではデータは共有されません。
- 機種変更、Safariデータ削除、ブラウザ変更に備えて、定期的に「JSONエクスポート」でバックアップしてください。
- `file://` で直接開くより、HTTPSのURLとして開く方が安定します。
